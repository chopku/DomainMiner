# Списки доменов для Gemini и OpenAI (domain_list)

Этот репозиторий содержит списки доменов, используемых популярными ИИ-сервисами (Gemini и OpenAI).

## Описание

Списки доменов предназначены для автоматического или ручного получения их IP-адресов. Это может использоваться для:
- Настройки маршрутизации (например, на роутерах Keenetic, OpenWrt для направления трафика через VPN).
- Фильтрации и настройки DNS-серверов.
- Обхода региональных ограничений.

## Содержимое

* `gemini.txt` — список доменов для сервисов Google Gemini, Google AI Studio, MakerSuite и смежных адресов.
* `openai.txt` — список доменов для сервисов OpenAI (ChatGPT, API, авторизация и вспомогательные домены).
